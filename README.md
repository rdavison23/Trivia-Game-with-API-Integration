# Trivia Game

Week 8: Build a Trivia Game with API Integration

A full‑stack trivia quiz application built with React on the frontend and Node.js + Express on the backend. Players choose the number of questions, category, and difficulty, then answer randomized multiple‑choice questions fetched from the Open Trivia Database API.

# Features

- Select number of questions, category, and difficulty

- Fetches trivia questions from a custom backend API

- Randomized answer order for each question

- Score tracking and game‑over summary

- Clean, responsive UI

- Fully resettable game flow

# Gameplay Flow

1. User selects game settings

1. Frontend requests questions from backend

1. Questions display one at a time

1. User selects an answer

1. Score updates

1. Game ends with a final score alert
